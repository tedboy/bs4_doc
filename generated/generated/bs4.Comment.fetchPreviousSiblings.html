13.1.4.23. bs4.Comment.fetchPreviousSiblings¶

Comment.fetchPreviousSiblings(name=None, attrs={}, text=None, limit=None, **kwargs)¶

Returns the siblings of this Tag that match the given criteria and appear before this Tag in the document.
